Overview4 features
#health-banner
overview

A color-coded status strip in Overview showing your overall network health as a 0–100 score. It aggregates the worst active alert state across all monitored systems so you spot critical issues at a glance.

USAGE
  1. Open Overview — the health strip appears below the KPI tiles.
  2. Green = clear, amber = warnings, red = critical breach.
  3. Click the banner to jump to Alerts and see firing rules.
#kpi-tiles
overview

Large summary cards showing top-line metrics — total subnets, DHCP leases, DNS zones, host uptime, and alert count. Each tile is clickable; delta badges show trend vs the previous period.

USAGE
  1. Click any KPI tile to jump to that section.
  2. Click a colored delta badge to navigate and pre-filter.
  3. Tiles auto-refresh every polling cycle.
#tactical-bar
overview

A 5-segment strip below the KPI tiles showing live counts for Hosts, Subnets, Zones, Alerts, and Feeds. Color-codes by severity so you read the network pulse in one row.

USAGE
  1. Find the strip between KPI tiles and the widget grid.
  2. Red = critical, amber = warnings, teal = healthy.
  3. Click a segment to navigate to its section.
IPAM / Subnets8 features
#subnet-table
ipam

A full list of all IP address pools with utilization bars showing how full each subnet is. Click any row to drill into details, DHCP lease history, and capacity forecast.

USAGE
  1. Go to IPAM / Subnets in the sidebar.
  2. Use the chip filter (All / Critical / Warning / OK).
  3. Click any subnet row to open the drill-down.
#capacity-forecast
ipam

Inside a subnet drill-down, a linear regression forecast shows how many days until the pool is exhausted at the current growth rate, based on historical snapshots in your browser.

USAGE
  1. Click any subnet row to open the drill-down panel.
  2. Scroll to the "Capacity Forecast" section.
  3. Days-to-full is red under 30, amber under 90.
Security3 features
#block-domains
security

One-click domain blocking via your Infoblox DNS Firewall. Add a domain to the block list directly from the dashboard and confirm the policy push without leaving the page.

USAGE
  1. Open Security and locate the Block Domains panel.
  2. Enter a domain and click Block.
  3. Use Unblock to remove an entry and re-push.
Alert Rules4 features
#threshold-alerts
alerts

Create custom alert rules that fire when any live metric crosses a threshold — e.g. "DHCP utilization above 90%". Rules evaluate every polling cycle and surface in the Health Banner.

USAGE
  1. Open Alert Rules and click "New rule".
  2. Pick a metric, operator, and threshold value.
  3. Save — the rule begins evaluating immediately.
Global Tools13 features
#command-palette
global

Press ⌘K (or Ctrl+K) to open a fuzzy-search command launcher. Type any section name, action, or feature to jump there instantly without touching the mouse.

USAGE
  1. Press ⌘K / Ctrl+K anywhere in the dashboard.
  2. Type a section, host, or action name.
  3. Press Enter on the highlighted result.

Overview

4 features
Health Banner

A color-coded status strip in Overview showing your overall network health as a 0–100 score. It aggregates the worst active alert state across all monitored systems so you spot critical issues at a glance.

How to use
  1. Open Overview — the health strip appears just below the KPI tiles.
  2. Green = all clear, amber = warnings, red = critical thresholds breached.
  3. Click the banner to jump to Alerts and see which rules fire.
KPI Tiles

Large summary cards showing top-line metrics — total subnets, DHCP leases, DNS zones, host uptime, and alert count. Each tile is clickable and navigates directly to its section. Delta badges show trend vs the previous period.

How to use
  1. Click any KPI tile to jump to that section.
  2. Click a colored delta badge (+3 subnets) to navigate and pre-filter.
  3. Tiles auto-refresh every polling cycle — no manual action.
Tactical Overview Bar

A 5-segment strip below the KPI tiles showing live counts for Hosts, Subnets, Zones, Alerts, and Feeds at a glance. Color-codes by severity so you can read the network pulse in one row.

How to use
  1. Find the strip between KPI tiles and the main widget grid.
  2. Red = critical, amber = warnings, teal = healthy.
  3. Click a segment to navigate to its section.

IPAM / Subnets

8 features
Subnet Table

A full list of all IP address pools with utilization bars showing how full each subnet is. Click any row to drill into that pool's details, DHCP lease history, and capacity forecast.

How to use
  1. Go to IPAM / Subnets in the sidebar.
  2. Use the chip filter (All / Critical / Warning / OK).
  3. Click any subnet row to open the drill-down panel.
Capacity Forecast

Inside a subnet drill-down, a linear regression forecast shows how many days until the pool is exhausted at the current growth rate, based on historical tile snapshots stored in your browser.

How to use
  1. Click any subnet row to open the drill-down panel.
  2. Scroll to the "Capacity Forecast" section inside the panel.
  3. Days-to-full is red under 30 days, amber under 90.

Security

3 features
Block / Unblock Domains

One-click domain blocking via your Infoblox DNS Firewall. Add a domain to the block list directly from the dashboard and confirm the policy push without leaving the page.

How to use
  1. Open Security and locate the Block Domains panel.
  2. Enter a domain and click Block.
  3. Use Unblock to remove an entry and re-push the policy.

Alert Rules

4 features
Define Threshold Alerts

Create custom alert rules that fire when any live metric crosses a threshold — e.g. "DHCP utilization above 90%". Rules evaluate on every polling cycle and surface in the Health Banner.

How to use
  1. Open Alert Rules and click "New rule".
  2. Pick a metric, operator, and threshold value.
  3. Save — the rule begins evaluating immediately.

Global Tools

13 features
Command Palette

Press ⌘K (or Ctrl+K) to open a fuzzy-search command launcher. Type any section name, action, or feature to jump there instantly without touching the mouse.

How to use
  1. Press ⌘K / Ctrl+K anywhere in the dashboard.
  2. Type a section, host, or action name.
  3. Press Enter on the highlighted result.
